# AI-Powered Restaurant Website

An AI-powered restaurant management system featuring:
- **FastAPI Backend**: Robust API with SQLAlchemy and PostgreSQL.
- **React Frontend**: Modern UI with Vite and Tailwind CSS.
- **AI Integration**: RAG-based Menu Assistant and Sentiment Analysis for reviews.
- **Payment Gateway**: Razorpay integration for seamless transactions.
- **Real-time Updates**: WebSockets for live order tracking.
